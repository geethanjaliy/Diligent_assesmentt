# Ecommerce Data Analysis Project

A complete data pipeline that generates synthetic ecommerce data, stores it in SQLite, and performs complex data analysis.

## Project Structure

### Steps:
1. **Step 1**: Generate synthetic ecommerce data (5 CSV files)
2. **Step 2**: Create SQLite database and import data
3. **Step 3**: Run complex multi-table queries and generate insights
4. **Step 4**: Push project to GitHub

### Files:
- `step1_generate_data.py` - Generates synthetic ecommerce data
- `step2_create_database.py` - Creates SQLite database and imports data
- `step3_run_queries.py` - Runs complex SQL queries for analysis
- `requirements.txt` - Python dependencies
- `*.csv` - Generated data files and query results
